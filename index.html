Ecovalia

Ecovalia is a tabletop board/card game about the development and conservation of reserve and national parks in South Australia.

How to get the game

We're currently working hard on getting the game ready to play. Eventually we plan to release the game here on this website as a print-and-play game. You will be able to download the PDF files for the game, print them out and get playing.
